# pysh

An interactive python shell that can be started to help with debug and prototyping.

# Setup lib
## Requirements
* ipython

## Install
```bash
pip install .
```

## Uninstall
```bash
pip uninstall pysh
```

## Add module to python path
An alternative to install is to add the environment variable PYTHONPATH.
```
~\projects\pysh\src
```
